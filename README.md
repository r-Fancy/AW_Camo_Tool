# AW_Camo_Tool

# Function

It copies one image multiple times, using the number of weapon names for the camo type you choose. It then renames the images to match that specific camo type.

# Usage

1. Run tools.py
2. Choose Between these options:
    B: basic_weapon
    N: non_animated 
    mk2
    MK15
3. Drag & Drop Image you want to copy.
4. Check the now created Output folder

# Aditional

To have the camos load make sure your file extension is PNG, as anything else is unsupported by the current clients.
